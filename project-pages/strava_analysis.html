Stava Analysis
To better understand where I ride my bike and go for runs, I created a heatmap!
Project Details / Background
I accessed my Strava data via their API and used my route data to create a heatmap using Python. I processed the data using GeoPandas and the Shapely library before creating the above map with Folium. Data is from 3-23-204 to 9-05-2024, most activities are rides but some are walks and runs.
